
## blog-api

The API for amazing blogs.

## Installation

```bash
$ yarn install
```

Create a file named `.env` based on `.env.example` and fulfill the variables.

## Running the app

```bash
docker-compose up
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
